# 69 x的平方根

easy

## 题目

实现int sqrt(int x)函数。

计算并返回x的平方根，其中x 是非负整数。

由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。

示例 1:
```
输入: 4
输出: 2
```
示例 2:
```
输入: 8
输出: 2
说明: 8 的平方根是 2.82842..., 
    由于返回类型是整数，小数部分将被舍去。
```

## 思路

#### 二分法

因为结果都是正整数，可以把结果从1到x之间进行二分法查询。

注意的点，有两个，一个是获取中间值时，需要注意不要超出int的范围。可使用位运算。

结果计算时，a*a=x时也会出现int溢出的情况，可以使用除法，来避免溢出的情况。
