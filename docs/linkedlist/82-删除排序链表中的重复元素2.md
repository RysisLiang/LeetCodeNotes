# 82 删除排序链表中的重复元素2

medium

## 题目

存在一个按升序排列的链表，给你这个链表的头节点 head ，请你删除链表中所有存在数字重复情况的节点，只保留原始链表中没有重复出现的数字。

返回同样按升序排列的结果链表。

示例 1：
```
输入：head = [1,2,3,3,4,4,5]
输出：[1,2,5]
```
```
示例 2：
输入：head = [1,1,1,2,3]
输出：[2,3]
```

提示：

- 链表中节点数目在范围 `[0, 300]` 内
- `-100 <= Node.val <= 100`
- 题目数据保证链表已经按升序排列

## 思路

思路还是比较简单的。主要的几个问题就是，指针的指向问题，很可能出现循环指向。还有就是head的边界问题需要注意。
